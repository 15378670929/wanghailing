vue基础填坑

在vue中尽量不使用箭头函数

Data

在vue中，data为保证是响应式数据，对象必须是纯粹的对象；

在组件中的data必须声明返回一个初始数据对象的函数

require

在vue的js中引入图片路径需要使用require()，否则js会默认看出是一串字符串，在标签中动态生成图片路径的时候，也需要使用require

ref

动态的创建ref,并通过该ref获取组件，在正常的动态创建ref，关键在于获取，获取时一定要用[0] ！！！ this.$ref[‘wrapper’+i][0]，

v-for

在v-for循环生成的list中的每个标签添加一个属性，需要使用this.$nextick()

computed和watch的区别

computed

是计算值，主要应用就是简化template里面双大括号的计算和处理props或$emit的传值，具有缓存性，计算属性会立即返回之前的计算结果，而不必再次执行函数

watch

是观察动作，主要应用监听props，$emit或本组件的值执行异步操作，无缓存性，页面重新渲染时值不变化也会执行

华为自带的浏览器position：fixed的兼容问题

将bottom：auto即可。

驼峰命名法和短横线命名法

在DOM里接受用短横线命名，在js读取的时候用驼峰读取：

接收：:c-msg=’msg’

props/读取：cMsg/this.cMsg

如果感觉有点麻烦可以全用小写。

事件总线（event bus）

keep-alive

匹配首先检查组件自身的 name 选项，如果 name 选项不可用，则匹配它的局部注册名称 (父组件 components 选项的键值)。匿名组件不能被匹配。

delimiters

作用：改变存文本插入分隔符
